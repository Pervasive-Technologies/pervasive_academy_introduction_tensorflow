To build and run this docker follow this steps:

Windows:
1.- Install Docker toolbox following https://docs.docker.com/toolbox/toolbox_install_windows/
2.- Open a terminal and run: docker run -p 8888:8888 --name tensorflow-udacity -it gcr.io/tensorflow/udacity-assignments:1.0.0
3.- Copy and paste the url from the terminal to a browser

Linux:
1.- Install Docker following https://docs.docker.com/engine/installation/linux/ubuntu/
2.- Open a terminal and run: docker run -p 8888:8888 --name tensorflow-udacity -it gcr.io/tensorflow/udacity-assignments:1.0.0
3.- Copy and paste the url from the terminal to a browser

IOS:
1.- Install Doker following https://docs.docker.com/docker-for-mac/install/
2.- Open a terminal and run: docker run -p 8888:8888 --name tensorflow-udacity -it gcr.io/tensorflow/udacity-assignments:1.0.0
3.- Copy and paste the url from the terminal to a browser
